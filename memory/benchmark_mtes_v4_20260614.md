# MTES v4 基准对比报告 — 2026-06-14

## 结果

MTES v4 被正式确认为**默认趋势判断策略**。

## 基准策略

基线策略（固定不变，用于对比参考）：
```
BULL = price > EMA200 AND EMA200斜率>0.001 AND DI+>DI- AND ADX>25
BEAR = price < EMA200 AND EMA200斜率<-0.001 AND DI->DI+ AND ADX>25
```

## 对比对象

| 策略 | 规则 | 类型 |
|------|------|------|
| **MTES v4** (LeanMTES.action_bias) | Ichimoku(60%) + EMA(40%) + ADX预过滤 + BOS/CHoCH | 后续开发重心 |
| EMA(50/200)黄金交叉 | 快均线上穿/下穿慢均线 | 仅参考（信号极稀疏） |
| MACD(12/26/9)零轴穿越 | MACD柱线正负穿越 | 参考（噪声较高） |
| SuperTrend(10,3) | ATR通道突破 | 不适合日线/周线 |

## 综合评价

- **方向正确性**: ✅ 与4条件基线一致率 76.7%，冲突率仅 0.1%
- **换手率**: ✅ 6.36% — 不高不低，恰到好处，避免低频不换和噪声切换两个极端
- **过滤震荡**: ✅ 中性率 ~55%，超过一半时间判断震荡不交易
- **跨市场稳健**: ✅ 72%~83% 一致率无大幅波动，从CN到US到HK都适用

**结论:** 默认使用 MTES v4 作为趋势判断策略。EMA_Cross/MACD/SuperTrend 仅在需要特定特征时做辅助参考。
