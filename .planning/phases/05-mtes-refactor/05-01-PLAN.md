## 技术细节

### 核心数据结构

```python
@dataclass(frozen=True)
class MajorTrendConfigV2:
    """MTES v2 配置"""
    asset_class: str = "stock"
    
    # 方向权重 (主要)
    direction_weight: float = 0.65
    
    # 强度组内部权重
    strength_components: dict[str, float] = None
    
    # 原始参数 (保留)
    adx_period: int = 14
    structure_window: int = 55
    swing_window: int = 20
    regime_window: int = 126
    momentum_windows: tuple[int, int, int] = (63, 126, 252)

@dataclass(frozen=True)
class MajorTrendResultV2:
    """MTES v2 结果"""
    # 核心输出
    trend_score: float        # -100 ~ +100
    direction: str            # "BULL" / "BEAR" / "NEUTRAL"
    confidence: float          # 0.0 ~ 1.0
    
    # 方向组
    direction_signal: float    # -100 ~ +100
    direction_confidence: float # 0.0 ~ 1.0
    
    # 强度组
    strength_score: float     # 0 ~ 100
    strength_components: dict[str, float]
    
    # 原始分项 (保留)
    raw_direction: float
    raw_strength: float
    raw_structure: float
    raw_momentum: float
    raw_regime: float
    raw_mtf: float
    
    # 元数据
    regime: str
    regime_flags: list[str]
    top_drivers: list[dict]
    explanation: str
    metadata: dict
```

### 新评分流程

```
输入: df (OHLCV)
   ↓
计算 6 维度原始分
   ↓
calculate_direction_signal → direction_signal + direction_confidence
   ↓
calculate_strength_score → strength_score (5 维度平均)
   ↓
calculate_final_score → trend_score + confidence
   ↓
classify_trend_state → trend_state
   ↓
输出: MajorTrendResultV2
```

### 功能验收
- [ ] trend_score 范围 -100 ~ +100
- [ ] direction 独立于 trend_score
- [ ] confidence 反映可信度

### 数值验收
- [ ] BULL_STRONG: score >= 60 AND direction = "BULL"
- [ ] BULL_CONFIRMED: 30 <= score < 60 AND direction = "BULL"
- [ ] BULL_EARLY: 0 <= score < 30 AND direction = "BULL"
- [ ] NEUTRAL_CHOPPY: score 接近 0
- [ ] BEAR_EARLY: -30 < score <= 0 AND direction = "BEAR"
- [ ] BEAR_CONFIRMED: -60 < score <= -30 AND direction = "BEAR"
- [ ] BEAR_STRONG: score <= -60 AND direction = "BEAR"
