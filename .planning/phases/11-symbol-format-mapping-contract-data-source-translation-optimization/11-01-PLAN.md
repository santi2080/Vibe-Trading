<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| user/watchlist/config → symbol contract | 用户输入 symbol 进入 canonical normalization；错误 coercion 会导致错误市场或错误供应商调用。 |
| canonical contract → vendor loader | canonical symbol 被翻译成 vendor-specific symbol；vendor symbol 只能存在于边界内。 |
| vendor loader → caller outputs | loader 结果重新进入项目内部；必须映射回 canonical key，不能污染 cache/run plan/report。 |
| fallback/unresolved reporting | primary/fallback 缺失结果必须以 canonical symbol 报告，不能丢失或显示 vendor key。 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|---|---|---|---|---|
| T-11-01 | Tampering | `SymbolTranslator` normalization | mitigate | Contract tests pin exact canonical normalization and unsupported combos; no silent suffix stripping. |
| T-11-02 | Information Disclosure | HybridDataFetcher output keys | mitigate | Remap vendor keys to canonical keys and assert no vendor-only keys in tests. |
| T-11-03 | Repudiation | fallback/unresolved symbols | mitigate | Preserve canonical unresolved semantics and run `test_get_market_data_unresolved.py`. |
| T-11-04 | Denial of Service | unsupported vendor/market combos | mitigate | Skip unsupported source/symbol combinations before provider call; per-symbol handling must not drop resolved symbols. |
| T-11-05 | Spoofing | Tushare CN futures fallback | mitigate | Mark Tushare CN futures unsupported unless a tested futures endpoint exists; do not route CN futures into stock daily endpoint. |
| T-11-SC | Tampering | package installs | accept | No package install is planned; no Package Legitimacy Audit required. |

## Risk Model

| Risk | Mitigation |
|---|---|
| HK four-digit vs five-digit duplicate cache keys | Plan locks five-digit canonical `.HK`; tests require `0700.HK -> 00700.HK`. |
| Direct loaders break because HybridDataFetcher now passes vendor symbols | HybridDataFetcher remaps output and accepts `raw.get(canonical)` compatibility fallback; direct loaders regression tests remain required. |
| Full detector unification expands scope | Add compatibility tests only; unify only if those tests prove a direct SYM-01/SYM-03 break. |
| yfinance proxy/live behavior masks contract quality | Required tests are mocked/offline; live smoke remains optional and not part of done. |
</threat_model>
