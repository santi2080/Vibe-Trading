# MTES v3 讨论日志

## 讨论摘要

### 核心问题识别

1. **MTES Direction 维度使用 SMA 而非 EMA**
   - 滞后性比预期更严重
   - 平均滞后 100-150 根 K 线
   - 比简单双 EMA 更慢

2. **200 EMA 小周期替代大趋势的根本缺陷**
   - 数学错位：M5-200EMA ≠ H4-200EMA
   - 滞后严重
   - 震荡失效（70% 时间）

### 解决方案决策

| 决策项 | 选择 | 理由 |
|--------|------|------|
| **演进方向** | 架构重构 | 现有架构无法解决滞后问题 |
| **使用场景** | 两者兼顾 | 批量扫描 + 精选分析 |
| **核心架构** | 分层递进 | 逻辑清晰，每层职责明确 |

### 三种进阶策略评估

#### A. SMC/ICT 市场结构法
- **核心**: 波段高低点序列 (HH/HL, LH/LL)
- **优势**: 无滞后，识别机构意图
- **适用**: 趋势判断

#### B. Elder 三重滤网
- **核心**: MACD 斜率 + RSI 极值 + Buy Stop
- **优势**: 顺势交易 + 逆势切入
- **适用**: 入场时机

#### C. 多维指标矩阵
- **核心**: Ichimoku + ADX + MACD
- **优势**: 互锁验证
- **适用**: 趋势确认

### 整合决策

采用**分层递进架构**：
- Layer 1: SMC + Elder 第一滤 + Ichimoku → 大周期趋势
- Layer 2: ADX + 背离 + Regime → 趋势强度
- Layer 3: FVG + RSI + Range Filter → 入场时机
