# Plan: 趋势指标回测系统

**Phase:** 02-01  
**Milestone:** Vibe-Trading Analysis Enhancements  
**Created:** 2026-05-30  
**Status:** completed

---

## 1. 目标

构建趋势指标回测系统，评估不同趋势判断策略（指标）在各市场的表现：
- **核心问题**: 哪个指标/策略最适合趋势判断？
- **决策支持**: 为不同市场选择最优趋势指标

---

### 2.3 数据周期

**简单起见，使用以下周期：**
- **1D（日线）**: 主要分析周期
- **1W（周线）**: 从 1D 重采样，用于多周期验证

**不包含：**
- 日内数据（实盘才用）

### 2.4 数据时间范围

- **建议范围**: 2020-01-01 至今
- **最小要求**: 2 年数据（250+ 日线 bars）
- **覆盖场景**: 牛市、熊市、震荡市

## 3. 测试指标列表

### 3.1 待测试指标

| 指标 | 说明 | 分类 |
|------|------|------|
| **TrendFusion** | 加权 4 指标趋势评分 | 综合型 |
| **TrendMagic** | CCI + ATR 趋势判断 | 震荡型 |
| **SuperTrend** | ATR 趋势带 | 趋势跟踪 |
| **RangeFilter** | 范围过滤器趋势 | 趋势确认 |
| **MTES** | Multi-Timeframe Ensemble System | 多周期 |
| **EMA_Cross** | EMA 金叉/死叉 | 经典型 |
| **SMA_Slope** | 均线斜率 | 简单型 |
| **ADX** | 平均趋向指数 | 趋势强度 |

### 3.2 评估维度

| 维度 | 说明 | 计算方法 |
|------|------|----------|
| **方向准确性** | 信号方向与实际趋势一致率 | 多周期对齐评分 |
| **信号领先性** | 信号是否领先于价格变动 | 均线偏离度 |
| **噪音过滤** | 对假信号的抑制能力 | 趋势内反向波动 |
| **稳定性** | 不同市场的一致性 | 跨品种平均得分 |

3. **建议**:
   - 日线趋势判断: 使用 RangeFilter
   - 周线趋势判断: 使用 SuperTrend 或 RangeFilter
   - 综合配置: 同时使用两者，信号一致时确认
