  <action>
    Create `agent/backtest/signals.py` with the following:

    1. `KeyNodeSignal` dataclass:
       - timestamp: pd.Timestamp
       - symbol: str
       - direction: Literal["BULL","BEAR","NEUTRAL"]
       - readiness: Literal["READY","WAIT","BLOCKED","EXHAUSTED","UNKNOWN"]
       - signal_score: float
       - components: dict[str, float] (per-source scores)
       - entry_action: Literal["OPEN","CLOSE","HOLD"]
       - reason: str
       - atr_trailing_stop: Optional[float] (current stop level)

    2. `CompositeSignalOutput` dataclass:
       - key_nodes: List[KeyNodeSignal]
       - per_source_signals: dict[str, List[dict]] (source_name -> TrendResult dicts)
       - metadata: dict[str, Any] (atr_period, atr_multiplier, config)

    3. `KeyNodeSignalRecorder` class:
       - `__init__(emit_all: bool = False)`: emit_all=True for debug, emits every bar
       - `_prev: Dict[str, TradingSignal]`: tracks last signal per symbol
       - `record(symbol, signal, action, atr_trailing_stop=None)`: stores KeyNodeSignal when direction or readiness changed
       - `_is_key_node(symbol, signal) -> bool`: True when direction!=prev.direction OR readiness!=prev.readiness OR prev is None
       - `to_dataframe() -> pd.DataFrame`: converts _records to DataFrame
       - `to_output(per_source: dict[str, Any]) -> CompositeSignalOutput`: full output
       - `reset()`: clears _prev and _records (for new backtest run)
       - `__len__() -> int`: number of records

    4. Key implementation detail: `action` parameter is the entry_action derived from D-01:
       - D-01: BULL+READY -> OPEN long, BEAR+READY -> OPEN short, else HOLD
       - If position was open and now flat -> CLOSE
       - If direction changed while position held -> CLOSE previous, OPEN new

    Import TradingSignal from `src.strategies.composite.base` for type hints only.
    Do NOT import CompositeTrendStrategy here - that belongs in composite_engine.py.
  </action>
