    Create `agent/backtest/engines/composite_engine.py` with three classes:

    1. **PositionManager** (D-02 trailing stop):
       - `__init__(atr_multiplier: float = 2.0, atr_period: int = 14)`
       - `_entry_high: Dict[str, float]`, `_entry_low: Dict[str, float]` (per symbol)
       - `_atr: Dict[str, float]` (current ATR per symbol)
       - `_direction: Dict[str, int]` (current position direction 1/-1/0)
       - `_signal_ema_high: Dict[str, float]`, `_signal_ema_low: Dict[str, float]` (per-bar trailing)
       - `update_entry(symbol, bar)`: sets entry_high=entry_low=bar["high"]=bar["low"] on open
       - `update_trailing(symbol, bar)`: tracks bar["high"]/bar["low"] as signal_ema_high/signal_ema_low
       - `update_atr(symbol, atr_value)`: stores current ATR
       - `check_exit(symbol, bar) -> tuple[bool, str]`: returns (should_exit, reason)
         - Long: low <= (signal_ema_high - atr_multiplier*atr) -> trailing_stop_long
         - Short: high >= (signal_ema_low + atr_multiplier*atr) -> trailing_stop_short
       - `close_position(symbol)`: clears entry/trailing/ATR state for that symbol
       - `_check_atr(symbol) -> bool`: True when ATR available for this symbol

    2. **CompositeBacktestSignalEngine**:
       - `__init__(composite: CompositeTrendStrategy, atr_multiplier: float = 2.0, atr_period: int = 14)`
       - `self._recorder: KeyNodeSignalRecorder` (initialized fresh)
       - `self._position_mgr: PositionManager`
       - `generate(data_map: Dict[str, pd.DataFrame]) -> Dict[str, pd.Series]`:
         - For each symbol, df:
           1. Compute ATR: df["atr"] = atr(df, self._atr_period)
           2. Call composite.analyze(df) -> TradingSignal
           3. Map direction+readiness to action: D-01
              - BULL+READY -> action="OPEN_LONG", direction_val=1
              - BEAR+READY -> action="OPEN_SHORT", direction_val=-1
              - else -> action="HOLD", direction_val=0
           4. Update position manager: update_atr, update_trailing
           5. Record key-node signal via recorder.record(symbol, signal, action)
           6. Return pd.Series([direction_val], index=[df.index[-1]])
         - Return dict mapping symbol -> Series

    3. **CompositeEngine** (extends BaseEngine for D-01/D-02 position management):
       - `__init__(config: dict, codes: list[str], composite: CompositeTrendStrategy, **engine_kwargs)`
       - Stores composite and position manager
       - Overrides `_rebalance()` to inject D-01 checks:
         - Before opening: check signal_map via _get_signal_action(symbol) -> if not READY, flat
         - Call super()._rebalance() with modified target
       - Overrides `_close_position()` to also call position_mgr.close_position(symbol)
       - Override `run_backtest()` to use composite signal_engine internally:
         - Creates CompositeBacktestSignalEngine from self.composite
         - Passes to super().run_backtest() as signal_engine
         - After run: extracts signals from signal_engine._recorder.to_output() -> writes artifacts
       - Overrides `_execute_bars()` to check trailing stop exits at each bar:
         - For each symbol with open position:
           - Check position_mgr.check_exit(symbol, bar)
           - If should_exit: call _close_position(symbol, bar["close"], timestamp, reason)

    Key constraint: CompositeBacktestSignalEngine.generate() MUST be called with full data_map
    so that _execute_bars in BaseEngine gets aligned dates. The generate() method processes
    ALL bars (not just last), enabling per-bar trailing stop tracking.

    CRITICAL: PositionManager must be reset on new backtest run (add reset() method).
    CompositeBacktestSignalEngine.__init__ creates fresh recorder + position_mgr each time.

    Pitfall to avoid (from RESEARCH): PositionManager state per symbol must not leak across symbols.
